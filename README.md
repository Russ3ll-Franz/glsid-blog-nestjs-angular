## Installing project

### Frontend folder

 1- Get The Last version 7 of Angular using npm
```
 npm install -g @angular/cli
```

  2- install dependencies in folder of client

```
  npm install
```
  3- Start the App 
  ```
  ng serve --open
```
 ### Backend folder
  1- Get The Last version of nestJs using npm:
```
 $ npm i -g @nestjs/cli
```

  2- install dependencies in the server folder

```
  npm install
```
  3- Start the App 
```
  npm run start
```
